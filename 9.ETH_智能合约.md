## what soldity.

面向对象

智能合约是运行在区块链上的一段代码，代码的逻辑定义了合约的内容。

=> 智能合约的账户保存了合约当前的运行状态 

  balance : 当前余额

  nonce : 交易次数

  code : 合约代码

  storage : 存储，数据结构是一棵MPT

  solidity 是只能合约最常用的语言，语法与JavaScript很接近
